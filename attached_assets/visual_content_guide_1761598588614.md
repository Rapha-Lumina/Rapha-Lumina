# VISUAL CONTENT DESIGN GUIDE
## Awakening to Consciousness Course

## BRAND IDENTITY

### Colour Palette

**Primary Colours:**
- Deep Purple: #4A3B6B (consciousness, depth, spirituality)
- Teal/Turquoise: #2A9D8F (clarity, flow, peace)
- Gold/Amber: #F4A259 (illumination, awakening, wisdom)

**Secondary Colours:**
- Soft Cream: #F7F3E9 (backgrounds, lightness)
- Deep Indigo: #1B1B3A (text, grounding)
- Rose Gold: #D4AF37 (accents, highlights)

**Gradient Options:**
- Purple to Teal (for transitions/consciousness flow)
- Deep Indigo to Gold (for awakening themes)
- Soft gradient overlays on images

### Typography

**Headings:**
- Font: Montserrat Bold or similar clean sans-serif
- Use for: Lesson titles, module headers, emphasis

**Body Text:**
- Font: Open Sans, Lato, or Raleway
- Use for: Descriptions, explanations, workbook text

**Accent/Quotes:**
- Font: Cormorant Garamond or similar elegant serif
- Use for: Pull quotes, wisdom statements, key phrases

### Visual Style
- Clean, modern, spacious
- Minimalist with intentional elements
- Nature-inspired where appropriate
- Ethereal/cosmic elements (subtle)
- Balance between grounded and transcendent

## KEY VISUAL ASSETS TO CREATE

### 1. THE BUILDING METAPHOR
**Concept:** 7-storey building representing the levels of consciousness

**Design Specifications:**
- Vertical illustration
- 7 distinct floors/levels
- Each level has unique colour
- Basement darker, top floor lighter
- Stairway visible connecting levels
- Window on each floor showing different "view"
- Minimalist, clean lines
- Can be animated (optional)

**Level Colours:**
1. Victim (Basement): Dark grey/brown
2. Warrior (Level 2): Red/orange
3. Achiever (Level 3): Gold/yellow
4. Intention (Level 4): Light purple
5. Flow (Level 5): Teal/aqua
6. Unity (Level 6): Soft pink/rose
7. Being (Level 7): White/light

**Usage:** Intro videos, course overview, workbook cover

---

### 2. THE RIVER METAPHOR
**Concept:** Flowing river representing life's current

**Design Specifications:**
- Overhead view or side profile
- Flowing river with clear current
- Boat in the river (small but visible)
- Person in boat (illustrated or symbolic)
- Gentle curves showing natural flow
- Colour: Teal water, natural banks
- Could show progression: rowing upstream → letting go → flowing

**Usage:** Lesson 9 (Flow), meditation visuals

## SLIDE DECK SPECIFICATIONS

### For Video Lessons

**Master Slide Template:**
- Background: gradient or solid course colour
- Title area: top third
- Content area: middle (spacious)
- Footer: lesson number + course name
- Font sizes: Heading 44pt, Body 28pt minimum

**Slide Types Needed:**

1. **Title Slide**
   - Lesson number + title
   - Visual element
   - No clutter

2. **Section Header**
   - Section name only
   - Accent colour
   - Minimal

3. **Content Slide**
   - Max 3-5 bullet points
   - Large, readable text
   - One concept per slide
   - Image optional

4. **Quote Slide**
   - Centered quote
   - No other elements
   - Elegant font
   - Breathing room

5. **Diagram/Chart**
   - Simple visuals
   - Clearly labelled
   - Not text-heavy
   - One diagram per slide

6. **Exercise Instructions**
   - Numbered steps
   - Clear, simple
   - Visual cues
   - "Pause here" indicator

7. **Key Takeaway**
   - Summary statement
   - Highlighted differently
   - Memorable phrasing
   - Can repeat

## ACCESSIBILITY CONSIDERATIONS

- **Colour Contrast:** Ensure text meets WCAG 2.1 AA standards
- **Font Size:** Minimum 16pt for body text
- **Alt Text:** Provide for all images
- **Captions:** Include for videos
- **Printability:** Black and white versions of workbook

## EXAMPLE VISUAL DESCRIPTIONS

### The Building Visual (Detailed)

**Scene:** Modern, minimalist 7-storey building in cross-section

**Basement (Level 1 - Victim):**
- Dark, cramped space
- Figure sitting with head in hands
- Small window showing limited, dark view
- Colour: Grey/brown tones
- Feeling: Heavy, confined

**Level 2 (Warrior):**
- Figure standing with fists clenched
- Red/orange colour scheme
- Symbols of battle (shield, sword decorative)
- Window shows battlefield
- Feeling: Tense, energized

**Level 3 (Achiever):**
- Figure at desk with laptop, charts
- Gold/yellow tones
- Trophies, awards visible
- Window shows cityscape, skyscrapers
- Feeling: Busy, accomplished

**Level 4 (Intention):**
- Figure meditating with clear intention
- Soft purple light
- Flowing energy patterns
- Window shows synchronistic symbols
- Feeling: Aligned, clear

**Level 5 (Flow):**
- Figure in graceful movement
- Teal/aqua colours
- Water-like flow patterns
- Window shows river, nature
- Feeling: Effortless, fluid

**Level 6 (Unity):**
- Figure with arms open, connected
- Rose/pink tones
- Interconnected circles/mandala
- Window shows web of connection
- Feeling: Expansive, loving

**Level 7 (Being):**
- Figure in pure stillness
- White/golden light
- Minimal, spacious
- Window shows infinite sky
- Feeling: Peaceful, vast

**Stairway:**
- Clearly visible connecting all levels
- Lit from above
- Open, inviting
- Emphasizes choice to ascend
